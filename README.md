# voxAfrica

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Contributions Welcome](https://img.shields.io/badge/contributions-welcome-brightgreen.svg?style=flat)](CONTRIBUTING.md)

**voxAfrica** is an open-source initiative dedicated to curating, developing, and sharing **speech datasets for Bantu and African languages**. This repository serves as a foundation for research in:

- Automatic Speech Recognition (ASR)
- Text-to-Speech (TTS)
- Multilingual Language Modeling
- African NLP Research

---

## Mission

To democratize African language technologies through open collaboration by collecting, organizing, and sharing high-quality speech datasets and linguistic resources that empower researchers, developers, and communities across the continent.

## Quick Start

### Prerequisites

- Python 3.8+
- Git
- (Optional) CUDA for GPU acceleration

### Installation

```bash
# Clone the repository
git clone https://github.com/SeViVI-Tese/voxAfrica.git
cd voxAfrica

# Install dependencies (when available)
pip install -r requirements.txt
```

## Available Datasets

See our comprehensive [Dataset Catalog](datasets/speech_datasets_links.md) for:

- Links to public African speech datasets
- Dataset descriptions and statistics
- Supported languages and dialects
- Licensing information

**Currently Featured Languages:**
- Shona, Ndebele (Zimbabwe)
- Swahili (Tanzania)
- Zulu, Xhosa (South Africa)
- Yoruba, Hausa, Igbo (Nigeria)
- And many more!
